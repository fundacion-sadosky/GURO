# GURO
La Fundación Sadosky desarrolla el kit GURO para actualizar los robots Robobloq Qoopers —ya desplegados en escuelas de todo el país— reemplazando el controlador propietario Qmind Plus por la BBC Micro:bit: una placa didáctica de hardware y software completamente abiertos, con una comunidad global activa de millones de docentes y estudiantes.
