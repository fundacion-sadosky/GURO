# Primeros pasos

Esta guía te ayuda a prepararte para el armado del robot GURO con el kit Robobloq Qoopers.

## Qué necesitás

- Kit GURO (micro:bit, placas adaptadoras)
- Kit Robobloq Qoopers (todas las piezas incluidas en la caja)
- Un destornillador Phillips
- Una computadora con puerto USB (para programar el robot)

## Inicio rápido

1. Desembalá todos los componentes y verificá la lista de piezas contra el inventario
2. Seguí la [Guía de usuario](user-guide/index.md) para las instrucciones de armado paso a paso
3. Una vez armado, conectá el robot por USB y cargá tu primer programa

> [!tip] ¿Primera vez?
> Comenzá con la [Guía de usuario](user-guide/index.md) — recorre cada paso del armado
> con fotos y diagramas.
