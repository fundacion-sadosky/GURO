# Contributing

Thank you for your interest in contributing!

This project accepts contributions from **anyone**, regardless of their technical background.

## Editing with Obsidian

The `docs/` folder is configured as an **[Obsidian](https://obsidian.md) vault** — a free Markdown notes editor. This allows editing the documentation without touching the terminal or the site code.

### Initial setup

1. Install [Obsidian](https://obsidian.md) (free)
2. Open Obsidian and select **Open vault → Open folder as vault**
3. Navigate to the `docs/` folder of this repository and confirm

### Recommended settings

Once the vault is open, configure in `Settings → Files and links`:

| Setting | Recommended value |
|---|---|
| Default location for new notes | Same folder as current file |
| Use `[[Wikilinks]]` | **Disabled** |
| New link format | Relative path to file |

> [!tip] Why disable wikilinks?
> Standard links (`[text](file.md)`) work both in Obsidian
> and in the published site. Wikilinks (`[[file]]`) only work in Obsidian.

### What you can do from Obsidian

- Create and edit `.md` pages directly
- View embedded images and Mermaid diagrams in preview
- Navigate between pages using links
- Use callouts (`> [!note]`, `> [!tip]`, etc.) that render on the site

Changes are saved as regular `.md` files: they are then uploaded to GitHub with a Pull Request like any other change.
