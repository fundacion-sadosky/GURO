# Contribuciones

¡Gracias por tu interés en contribuir!

Este proyecto acepta contribuciones de **cualquier persona**, independientemente de sus conocimientos técnicos.

## Edición con Obsidian

La carpeta `docs/` está configurada como un **vault de [Obsidian](https://obsidian.md)** — un editor de notas Markdown gratuito. Esto permite editar la documentación sin tocar la terminal ni el código del sitio.

### Configuración inicial

1. Instalá [Obsidian](https://obsidian.md) (gratuito)
2. Abrí Obsidian y seleccioná **Abrir vault → Abrir carpeta como vault**
3. Navegá hasta la carpeta `docs/` de este repositorio y confirmá

### Ajustes recomendados

Una vez abierto el vault, configurá en `Configuración → Archivos y enlaces`:

| Ajuste | Valor recomendado |
|---|---|
| Ubicación predeterminada para nuevas notas | Misma carpeta que el archivo actual |
| Usar `[[Wikilinks]]` | **Desactivado** |
| Formato de nuevos enlaces | Ruta relativa al archivo |

> [!tip] ¿Por qué desactivar los wikilinks?
> Los enlaces estándar (`[texto](archivo.md)`) funcionan tanto en Obsidian
> como en el sitio publicado. Los wikilinks (`[[archivo]]`) solo funcionan en Obsidian.

### Lo que podés hacer desde Obsidian

- Crear y editar páginas `.md` directamente
- Ver imágenes embebidas y diagramas Mermaid en la previsualización
- Navegar entre páginas con enlaces
- Usar callouts (`> [!note]`, `> [!tip]`, etc.) que se renderizan en el sitio

Los cambios se guardan como archivos `.md` normales: luego se suben a GitHub con un Pull Request como cualquier otro cambio.
