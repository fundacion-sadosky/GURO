# Getting started

This guide will help you prepare for assembling the GURO robot with the Robobloq Qoopers kit.

## What you need

- GURO kit (micro:bit, adapter boards)
- Robobloq Qoopers kit (all parts included in the box)
- A Phillips screwdriver
- A computer with a USB port (to program the robot)

## Quick start

1. Unpack all components and check the parts list against the inventory
2. Follow the [User guide](user-guide/index.md) for step-by-step assembly instructions
3. Once assembled, connect the robot via USB and load your first program

> [!tip] First time?
> Start with the [User guide](user-guide/index.md) — it walks through each assembly step
> with photos and diagrams.
